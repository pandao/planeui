Plane UI

HTML5 跨终端响应式前端界面框架

The Modern HTML5 Cross-Device Responsive Front-end UI Framework.

最新版本：v0.1.0 预览版，更新于：2015-06-26

主要特性

移动优先

遵循 Mobile first （移动优先）设计理念，从小到大自适应屏幕。

模组化

自带多种常用组件，模块化设计，方便快速地构建应用。

优雅降级

对浏览器进行分级支持，可以对IE8 提供最基本的兼容支持。

持续演进

向 Web Components 逐渐演进，提供更多 HTML5 + CSS3 + ES6 特性的支持。
